Avanza logotypen kan användas i flera storlekar. När logotypen visas i fullstorleks så anger storleken endast höjd medan bredden varierar.

| Storlek     | Small    | Medium   |
| ----------- | -------- | -------- |
| smallest    | 12px     | 22px     |
| smaller     | 14px     | 24px     |
| (default)   | 16px     | 28px     |
| larger      | 18px     | 30px     |
| largest     | 20px     | 32px     |
